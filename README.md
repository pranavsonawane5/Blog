# 📝 MERN Blog Application

A full-stack blog application built using the MERN stack (MongoDB, Express, React, Node.js) where users can register, login, create posts, edit posts, and view blogs with images.

---

## 🚀 Live Demo

- Frontend: https://blog-six-rosy-89.vercel.app/
- Backend: https://blog-backend-a27x.onrender.com

---

## 📌 Features

- 🔐 User Authentication (Register & Login using JWT)
- ✍️ Create, Edit, Delete Blog Posts
- 🖼️ Image Upload using Multer
- 📄 Rich Text Editor (TipTap)
- 🧑‍💻 Author-based Post Editing (Only author can edit)
- 🍪 Cookie-based authentication
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- Vite
- React Router
- TipTap Editor
- CSS

### Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Multer (file upload)
- bcrypt (password hashing)
